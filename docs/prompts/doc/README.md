# Prompt Reference Docs

Store prompt-engineering reference material here.

Use this directory for:

- Provider prompting guidance
- Agentic workflow patterns
- DriftSentinel-specific prompt conventions
- Prompt review checklists and heuristics

Keep executable task prompts out of this directory. Use `../draft/` for work-in-progress prompts and `../enhanced/` for validated prompts.

## Current Inventory

- `agentic-prompt-engineering.md`: prompt-level taxonomy, reusable sections, and system-vs-user prompt guidance
- `anthropic-prompting-best-practices.md`: Claude 4.x prompting guidance, tool-use patterns, and state-tracking advice
