# prime

Orient to the DriftSentinel repository before acting. Reads key governance docs
and reports understanding.

## Run

```
git ls-files
```

## Read

- `CLAUDE.md`
- `AGENTS.md`
- `README.md`
- Top-level `specs/` index

## Report

After reading, output a brief orientation summary:

- Product scope and current phase
- Repository taxonomy and key surfaces
- Methodology precedence
- Quality-control integration status (Codacy, Codecov, Snyk)
- Key files relevant to the current task context
- Any open questions about scope or intent

Do not make any changes during priming.
